# Introduction

Vue.js와 SpringBoot를 연동한 SPA 커뮤니티


- 개발 기간 : 21.11.01 - 21.11.17
- 팀 구성 : 개인 프로젝트(1명)
- 주요 기능 : 회원 및 게시판 CRUD, 추천, 댓글
- 개발 기술 : Vue.js(vuex, vue-router), JavaScript, Java, Element ui, MySQL, MyBatis
- 형상 관리 : git/github

# Features

**User**
- 회원가입(Bcrypt), 로그인(JWT)
- 마이 페이지

**Board**
- 인기/자유 게시판
- 검색(SSR, CSR)
- Pagination
- 댓글
- 추천

**Admin**
- 회원 관리
- 게시판 관리
